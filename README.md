## 프로젝트 개요 🎤

### Q. 무슨 프로젝트에요? 🤔

A. '상표의 신'이라는 기업에서 새롭게 출시하는 '특허의 신' 사이트 출시를 앞두고 특허 가능성 검토 신청서의 '입력 시간 단축' 문제를 해결해보는 프로젝트였어요 :)

### Q. 문제를 어떻게 해결했나요? 💡

A. 기존 제공받은 와이어프레임에는 모든 질문이 한 페이지에 모여있었는데요, 'one page, one thing'으로 한 화면에 입력받는 질문의 수를 최소화하여 사용자가 입력하는데 피로감을 덜 느끼도록 기획하였습니다.

### Q. 기능적으로 해결한 부분은 무엇인가요? 🔧

A. 개발 기간이 5일로 매우 짧다보니 많은 기능을 기획하고 개발하지는 못했지만 크게는 아래 4가지의 기능이 있어요.

1. **페이지를 이동해도 세션이 닫히지 않으면 입력 정보가 저장되어 안전하게 입력할 수 있습니다.**
2. **사용자가 원하는 입력 키워드를 선택하고 입맛에 맞게 양식을 작성할 수 있도록 토글 / 선택 입력창을 구현했습니다.**
3. **진행 상황을 status bar로 표시하여 막연함을 느끼지 않도록 구현하였습니다.**
4. **반응형으로 제작하여 모바일에서도 간단하게 특허 검토 신청을 할 수 있도록 하였습니다.**
